Classical ML

Logistic Regression TF-IDF Binary

Binary classification using Logistic Regression trained on TF-IDF sparse vectors derived from Hindi text tokens. Labels are merged into Pure (Legitimate) vs Septic (Synthetic). Uses distiluse-base-multilingual-cased embeddings for feature extraction.

scikit-learn SentenceTransformer LabelEncoder
Embedding
distiluse-base-multilingual-cased
Classifier
LogisticRegression (max_iter=1000)
Split
80% train / 20% test, stratified
Labels
Pure (Legitimate) · Septic (Synthetic)
Output Format
Accuracy · Classification Report · Confusion Matrix (PNG)

The production deployed model. Uses paraphrase-multilingual-MiniLM-L12-v2 to generate 384-dim dense embeddings from Hindi text, then trains a LogisticRegressionCV with 5-fold cross-validation for automatic hyperparameter selection.

scikit-learn sentence-transformers Cross-Validation ★ Production

03
Deep Learning

Artificial Neural Network TF-IDF Features

A fully connected feedforward neural network built with TensorFlow/Keras. Takes TF-IDF vectors (15,000 features, 1–2 n-grams) as input and learns non-linear decision boundaries through two hidden layers with dropout regularization.

TensorFlow Keras TF-IDF Adam
Architecture
Dense(256, ReLU) → Dropout(0.4) → Dense(128, ReLU) → Dropout(0.3) → Dense(1, Sigmoid)
Features
TF-IDF (15K features, 1-2 ngrams, sublinear_tf)
Training
10 epochs, batch_size=32, 10% val split
Loss
Binary Crossentropy, Optimizer: Adam (lr=0.001)
Output Format
Accuracy · Classification Report · Confusion Matrix Heatmap
ann_model.py
04
Deep Learning

Convolutional Neural Network Text CNN

1D CNN for Hindi text classification. Maps token sequences to dense 100-dim embeddings via a trainable Embedding layer, then applies Conv1D filters to detect n-gram-like patterns, followed by global max pooling to extract most prominent features.

TensorFlow Keras Tokenizer Conv1D GlobalMaxPool
Architecture
Embedding(10K, 100) → Conv1D(128, 5, ReLU) → GlobalMaxPool → Dropout(0.5) → Dense(64) → Dense(2, Softmax)
Vocab & Padding
Vocab: 10,000 tokens, max_length: 100 tokens
Training
100 epochs, batch_size=32
Loss
Categorical Crossentropy (2 classes: Softmax)
Output Format
Test Accuracy · Detailed Classification Report
cnn_binary.py

Long Short-Term Memory Sequence Model

A recurrent sequence model that processes tokenized Hindi text word-by-word, capturing long-range semantic dependencies via LSTM cells. Also plots training/validation accuracy and loss curves for in-depth analysis in the research paper.

TensorFlow LSTM Keras Tokenizer Seaborn
Architecture
Embedding(20K, 128) → LSTM(128) → Dropout(0.3) → Dense(64, ReLU) → Dropout(0.3) → Dense(1, Sigmoid)
Sequence
max_words: 20,000 | max_len: 50 | OOV token supported
Training
6 epochs, batch_size=32, 10% val split
Plots
Confusion Matrix + Accuracy/Loss training curves
Output Format
Accuracy · Classification Report · Confusion + Training Curves (PNG)
lstm_model.py

Ensemble LR + ANN Soft-Vote Averaging

A hybrid ensemble that averages the probability outputs of a Logistic Regression and a deep ANN, both trained on TF-IDF features (20K features, 1-3 n-grams). Soft voting combines the complementary strengths of both classifiers for improved robustness.

TensorFlow scikit-learn Soft Voting TF-IDF (20K)
Base Learners
LogisticRegression + ANN (Dense 256→128→1)
Features
TF-IDF (20K features, ngrams 1-3, sublinear_tf)
Combination
ensemble_probs = (lr_probs + ann_probs) / 2
Decision
Threshold at 0.5 for final binary prediction
Output Format
Ensemble Accuracy · Classification Report · Green-themed Confusion Matrix
ensemble_lr_ann.py

Instance-based learner that classifies Hindi news text by majority vote from its 5 nearest neighbors in TF-IDF feature space. Simple non-parametric baseline that reveals how well news authenticity separates in sparse vector space.

scikit-learn TF-IDF KNeighborsClassifier
Features
TF-IDF vectors (default vocabulary)
Hyperparams
n_neighbors=5, metric: Euclidean (default)
Split
80% train / 20% test, stratified
Labels
Legitimate (0) · Synthetic (1)
Output Format
Accuracy · Classification Report · Confusion Matrix Heatmap
knn_binary.py

Classical ML

Naïve Bayes Multinomial TF-IDF

A probabilistic generative model based on Bayes' theorem with feature independence assumption. MultinomialNB is well-suited for TF-IDF count-like features. Fastest to train among all classifiers — serves as the fastest baseline for comparison.

scikit-learn MultinomialNB TF-IDF
Algorithm
Multinomial Naïve Bayes (counts → likelihood)
Features
TF-IDF (default vocabulary, all n-grams)
Split
90% train / 10% test, stratified
Advantage
Fastest training, interpretable word probabilities
Output Format
Accuracy · Classification Report · Confusion Matrix Heatmap
naive_bayes_binary.py

Voting Ensemble 6-Model Hard Vote

A meta-ensemble using majority (hard) voting from 6 diverse TF-IDF classifiers: LinearSVC, MultinomialNB, KNN (k=5), RandomForest (200 trees), DecisionTree, and Logistic Regression. Features are TF-IDF with 5,000 features and 1-2 n-grams.

TF-IDF (5K features, 1-2 ngrams)
Voting
Hard voting (majority class wins)
Goal
Higher robustness by aggregating diverse classifiers
Output Format
Accuracy % · Classification Report · Confusion Matrix (text)
